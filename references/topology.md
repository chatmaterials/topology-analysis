# Topology Notes

- A compact invariant summary is useful, but it is not a substitute for a full topological diagnosis.
- Wilson-loop winding is often used as supporting evidence rather than a stand-alone conclusion.
- Surface spectra and arc-like features should be discussed together with the underlying model and symmetry context.
- Candidate ranking is useful for screening, but a convincing topological diagnosis still requires symmetry context and backend-specific validation.
